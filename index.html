youtube-dl extensions

Extensions for different browsers, see Github repo.

Download the extension:

  • Safari: Download from 0.1.1 and install by double-clicking it.
  • Chrome:
    1. Download from 0.1.1.
    2. Open Chrome and browse to the extensions page (chrome://extensions/).
    3. Drag the downloaded file to the page.

Use a local server

Both extensions support using a localserver for getting the URLs. It allows to get the correct links for sites like youtube.

Follow the steps to use it:

  1. Install the server from youtube-dl-web.
  2. Run python youtube_dl_sever.py.
  3. Turn on the option Use localserver in the extension's option page.
